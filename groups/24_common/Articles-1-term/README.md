### Здесь будут лежать ваши творческие! 

Чтобы сдать творческую надо создать PR (как это сделать отвечает [первая ссылка в интернете](https://jeka.by/post/1053/github-send-PR-to-other-repository/)) c папкой с вашей фамилией: внутри два файла - исходник (`.tex`) и `.pdf`. Исключение для `markdown`, там достаточно просто приложить `.md` файл.
